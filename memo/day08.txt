	- onload : 웹 브라우저가 열렸다면 로드 된 이후에 실행하는 로직 
	onload에 함수를 정의하는 방식은 브라우저 엔진에게 부담을 발생시킨다. 
	따라서 onload는 명확히 해당 브라우저가 실행되고 나서 실행되어야 하는 로직을 작성해야 한다. 
		- window.onload = ( ) => { }
		- window.addEventlistener("load",( ) =>{
			로직
		  })
	- resize : 웹 페이지 사이즈가 조정되었을 떄를 감지하여 사용할 수 있는 로직들을 구현한다. 퍼블리싱 과정에서 많이 사용된다.
